# Project Overview

Este projeto foi desenvolvido utilizando uma stack moderna com foco em performance, escalabilidade e facilidade de manutenção. A aplicação foi construída com React, TypeScript, Vite e Tailwind CSS, além da utilização de componentes do shadcn-ui para garantir consistência visual e produtividade durante o desenvolvimento.

# Tecnologias Utilizadas

React

TypeScript

Vite

Tailwind CSS

shadcn-ui

Essas tecnologias foram escolhidas pela ótima experiência de desenvolvimento e pela capacidade de criar interfaces rápidas, modernas e bem estruturadas.

# Arquitetura e Organização do Projeto

O código foi estruturado para manter clareza e fácil evolução:

src/
 ├─ components/     # Componentes reutilizáveis
 ├─ hooks/          # Hooks personalizados
 ├─ lib/            # Funções utilitárias
 ├─ pages/ or views/# Telas principais
 ├─ styles/         # Estilos e configurações do Tailwind
 └─ main.tsx        # Ponto de entrada da aplicação


A separação por responsabilidade facilita a manutenção e amplia possibilidades de crescimento do projeto.

# Como Executar o Projeto

Para rodar o projeto localmente:

* 1. Clonar o repositório
git clone <YOUR_REPOSITORY_URL>

* 2. Entrar no diretório
cd <YOUR_PROJECT_NAME>

* 3. Instalar dependências
npm install

* 4. Executar o servidor de desenvolvimento
npm run dev
